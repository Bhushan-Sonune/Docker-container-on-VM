# Deploy-Docker-container-on-VM
This project demonstrates how to deploy a Docker container on a cloud virtual machine (VM). It covers the basic steps of setting up a cloud server, installing Docker, pulling or building a Docker image, and running the containerized application so it can be accessed over the internet. This project is useful for understanding the fundamentals of cloud deployment, containerization, and application hosting.

## Steps Involved
1. Launch a cloud virtual machine (VM)
2. Connect to the VM using SSH
3. Update the server packages
4. Install Docker on the VM
5. Pull or build the Docker image
6. Run the Docker container
7. Configure security group or firewall rules
8. Expose the required application port
9. Access the application through the public IP
